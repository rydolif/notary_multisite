Бриф
1. Создать мульти-сайт на wordpress (по макету из figma).
2. Адаптивность.
3. Должна быть возможность отключать / включать блоки для отдельных подсайтов.
4. Для блоков «важное объявление», «новости нотариуса», «контактная информация» должна быть возможность редактирования из отдельной админки (не основной, пользователи должны быть отдельными), остальные блоки должны быть общими и редактироваться из основной админки. 
5. Блок «меню» должен содержать якоря на каждый из блоков + должна быть возможность включения / отключения “липкого” меню, меню должно формироваться исходя из показанных блоков на странице.
6. Кнопка «записаться» должна пролистывать страницу до формы «отправка документов».
7. Блок «важное объявление» должен быть редактируемым.
8. Блок «новости нотариуса» должен вести на отдельную страницу с записью.
9. Блок «нотариальные действия» должен содержать материалы в виде карусели, при этом должна быть возможность добавлять / удалять / редактировать нотариальные действия, при нажатии на ссылку «перечень необходимых документов» должно открываться pop-up окно с заданной записью.
10. Блок «тарифы» должен быть общим для всех сайтов. 
11. Для каждого блока «контактная информация» должна быть возможность редактирования информации и привязки к определенному адресу (я.карта).
12. Блок «отправка документов» должен отправлять письмо по заданному email (редактируется самостоятельно) + каптча.
13. Блок «публичные реестры» - карусель + возможность добавлять / удалять / редактировать публичные реестры (ссылки на сайты) и иконки к ним.
14. Минимальное количество плагинов.


Опционально: 
Сайт должен, по возможности, содержать микро-разметку + оптимизирован для PageSpeed Insights.
